# AI-SDR Assistant

## Setup Instructions

1. **Clone the repository:**
   ```bash
   git clone https://github.com/skhan-21/ai-sdr-assistant.git
   cd ai-sdr-assistant
   ```

2. **Install dependencies:**
   Make sure you have all required dependencies installed. Use the following command:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application:**
   After installing dependencies, you can run the application by using:
   ```bash
   python main.py
   ```
